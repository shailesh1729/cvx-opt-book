Bibliographic Notes¶

Following is a partial list of books and articles which have been referenced heavily in this work. This list is by no means exhaustive.

  • General introduction to optimization can be found in [12].

  • Main references for convex analysis are [5, 14].

  • [7] is a standard textbook for convex optimization theory, applications and algorithms.

  • [11] is a good reference for linear programming.

  • [6] covers alternating direction method of multipliers (ADMM) algorithms.

  • [13] provides good coverage on proximal algorithms.
