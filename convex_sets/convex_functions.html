6.6. Convex Functions¶

Throughout this section, we assume that \(\VV, \WW\) are real vector spaces. Wherever necessary, they are equipped with a norm \(\| \cdot \|\) or an real inner product \(\langle \cdot, \cdot \rangle\). They are also equipped with a metric \(d(\bx, \by) = \| \bx - \by \|\) as needed.

6.6.1. Convexity of a Function¶

Definition 6.41 (Convex function)

Let \(\VV\) be a real vector space. A real valued function \(f: \VV \to \RR\) is convex if \(\dom f\) is a convex set and for all \(\bx_1,\bx_2 \in \dom f\), and \(t \in [0, 1]\), we have:

(6.1)¶\[f(t \bx_1 + (1-t) \bx_2) \leq t f(\bx_1) + (1-t) f(\bx_2).\]

Fig. 6.1 Graph of a convex function. The line segment between any two points on the graph lies above the graph.¶

For a convex function, every chord lies above the graph of the function.

Definition 6.42 (Strictly convex function)

Let \(\VV\) be a real vector space. A convex function \(f: \VV \to \RR\) is strictly convex if for all \(\bx_1,\bx_2 \in \dom f\), where \(\bx_1\) and \(\bx_2\) are distinct, and \(t \in (0, 1)\), we have:

\[ f(t \bx_1 + (1-t) \bx_2) < t f(\bx_1) + (1-t) f(\bx_2). \]

In other words, the inequality is a strict inequality whenever the point \(\bx = t \bx_1 + (1-t) \bx_2\) is distinct from \(\bx_1\) and \(\bx_2\) both.

Definition 6.43 (Concave function)

We say that a function \(f\) is concave if \(-f\) is convex. A function \(f\) is strictly concave if \(-f\) is strictly convex.

Example 6.23 (Linear functional)

A linear functional on \(\VV\) in an inner product space has the form of an inner product parameterized by a vector \(\ba \in \VV\) as

\[ f_{\ba} (\bx) \triangleq \langle \bx, \ba \rangle \Forall \bx \in \VV. \]

i.e. the inner product of \(\bx\) with \(\ba\).

Theorem 6.64

All linear functionals on a real vector space are convex as well as concave.

Proof. For any \(\bx, \by \in \VV\) and \(t \in [0,1]\):

\[\begin{split} f_{\ba} (t \bx + (1-t) \by) &= \langle t \bx + (1-t) \by, \ba \rangle \\ &= t \langle \bx, \ba \rangle + (1-t)\langle \bx, \ba \rangle \quad \text{inner products are linear}\\ &= t f_{\ba} (\bx) + (1-t) f_{\ba} (\by). \end{split}\]

Thus, \(f_{\ba}\) is convex. We can also see that \(-f_{\ba}\) is convex too. Hence, \(f_{\ba}\) is concave too.

Example 6.24 (Arithmetic mean is convex and concave)

Let \(f : \RR^n \to \RR\) be given by:

\[ f(\bx) = \frac{x_1 + \dots + x_n}{n}. \]

Arithmetic mean is a linear function. In fact

\[\begin{split} f(\alpha \bx + \beta \by) &= \frac{1}{n} \left (\sum_{i=1}^n (\alpha x_i + \beta y_i) \right )\\ &= \alpha \frac{x_1 + \dots + x_n}{n} + \beta \frac{y_1 + \dots + y_n}{n}\\ &= \alpha f(\bx) + \beta f(\by) \end{split}\]

for all \(\alpha,\beta \in \RR\).

Thus, for \(t \in [0,1]\)

\[ f(t \bx + (1-t) \by) = t f(\bx) + (1-t) f(\by). \]

Thus, arithmetic mean is both convex and concave.

Example 6.25 (Affine functional)

An affine functional is a special type of affine function which maps a vector from \(\VV\) to a scalar in the associate field \(\FF\).

On real vector spaces, an affine functional on \(\VV\) is defined in terms of a vector \(\ba \in \VV\) and a scalar \(b \in \RR\) as

\[ f_{\ba, b} (\bx) \triangleq \langle \bx, \ba \rangle + b \Forall \bx \in \VV \]

i.e. the inner product of \(\bx\) with \(\ba\) followed by a translation by the amount of \(b\).

Theorem 6.65

All affine functionals on a real vector space are convex as well as concave.

Proof. A simple way to show this is to recall that for any affine function \(T\),

\[ T (t \bx + (1 - t) \by) = t T(\bx) + (1 - t) T(\by) \]

holds true for any \(\bx, \by \in \VV\) and \(t \in \FF\).

In the particular case of real affine functionals, for any \(\bx, \by \in \VV\) and \(t \in [0,1]\), the previous identity reduces to:

\[ f_{\ba, b} (t \bx + (1 - t) \by) = t f_{\ba, b}(\bx) + (1 - t) f_{\ba, b}(\by). \]

This establishes that \(f_{\ba, b}\) is both convex and concave.

Another way to prove this is by following the definition of \(f_{\ba, b}\) above:

\[\begin{split} f_{\ba, b} (t \bx + (1 - t) \by) &= \langle t \bx + (1 - t) \by, \ba \rangle + b\\ &= t \langle \bx, \ba \rangle + (1-t)\langle \bx, \ba \rangle + b\\ &= t \langle \bx, \ba \rangle + (1-t)\langle \bx, \ba \rangle + tb + (1-t)b\\ &= (t \langle \bx, \ba \rangle + + tb) + ((1-t)\langle \bx, \ba \rangle + (1-t)b)\\ &= t (\langle \bx, \ba \rangle + b) + (1-t)(\langle \bx, \ba \rangle + b)\\ &= t f_{\ba, b}(\bx) + (1-t) f_{\ba, b} (\by). \end{split}\]

Example 6.26 (Absolute value is convex)

Let \(f : \RR \to \RR\) be:

\[ f(x) = |x|. \]

with \(\dom f = \RR\).

Recall that \(|x|\) is a norm on the real line \(\RR\). Thus, it satisfies the triangle inequality:

\[ |x + y | \leq |x | + |y| \Forall x, y \in \RR. \]

In particular, for any \(t \in [0,1]\)

\[ |t x + (1-t)y | \leq |t x | + | (1- t) y| = |t| | x| + |1 - t| | y | = t |x| + (1-t) | y |. \]

Thus \(f\) satisfies the convexity defining inequality (6.1):

\[ f(t x + (1-t) y) \leq t f(x) + (1- t) f(y) \Forall x, y \in \RR. \]

Hence, \(f\) is convex.

Theorem 6.66 (All norms are convex)

Let \(\| \cdot \| \to \RR\) be a norm on a real vector space \(\VV\). Then, it satisfies the triangle inequality:

\[ \|\bx + \by \| \leq \|\bx \| + \|\by\| \Forall \bx, \by \in \VV. \]

In particular, for any \(t \in [0,1]\)

\[ \|t \bx + (1-t)\by \| \leq \|t \bx \| + \| (1- t) \by\| = |t| \| \bx \| + |1 - t| \| \by \| = t \|\bx \| + (1-t) \| \by \|. \]

Thus \(f\) satisfies the convexity defining inequality (6.1). Hence, \(f\) is convex.

Example 6.27 (Max function is convex)

Let \(f : \RR^n \to \RR\) be:

\[ f(\bx) = \max \{x_1, \dots, x_n \}. \]

with \(\dom f = \RR^n\).

Let \(\bx, \by \in \RR^n\) and \(t \in [0,1]\).

  1. Let \(x = f(\bx) = \max \{ x_1, \dots, x_n \}\) and \(y = f(\by) = \max \{ y_1, \dots, y_n\}\).

  2. Then, for every \(k=1,\dots,n\), we have: \(x_k \leq x\) and \(y_k \leq y\).

  3. Thus, \(t x_k + (1-t) y_k \leq t x + (1-t) y\).

  4. Taking maximum on the left hand side, we obtain:

    \[ \max_{k=1}^n (t x_k + (1-t) y_k) \leq t x + (1-t) y. \]

  5. But \(f(t \bx + (1-t) \by ) = \max_{k=1}^n (t x_k + (1-t) y_k)\).

  6. Thus, \(f\) satisfies the convexity defining inequality (6.1):

    \[ f(t \bx + (1-t) \by ) \leq t f(\bx) + (1-t) f(\by). \]

  7. Thus, \(f\) is convex.

Example 6.28 (Geometric mean is concave)

Let \(f : \RR^n \to \RR\) be given by:

\[ f(\bx) = \left ( \prod_{i=1}^n x_i \right )^{\frac{1}{n}} \]

with \(\dom f = \RR^n_{++}\).

Recall from AM-GM inequality:

\[ \left (\prod_{i=1}^n a_i \right )^{\frac{1}{n}} \leq \frac{1}{n} \left ( \sum_{i=1}^n a_i \right ). \]

  1. Let \(\bx, \by \in \RR^n_{++}\) and \(t \in [0,1]\).

  2. Let \(\bz = t \bx + (1-t) \by\).

  3. Note that \(\bz \in \RR^n_{++}\) too, hence every component of \(\bz\) is positive.

  4. With \(a_i = \frac{x_i}{z_i}\), and applying AM-GM inequality:

    \[ \frac{f(\bx)}{f(\bz)} = \left ( \prod_{i=1}^n \frac{x_i}{z_i} \right )^{\frac{1}{n}} \leq \frac{1}{n} \left ( \sum_{i=1}^n \frac{x_i}{z_i} \right ). \]

  5. With \(a_i = \frac{y_i}{z_i}\), and applying AM-GM inequality:

    \[ \frac{f(\by)}{f(\bz)} = \left ( \prod_{i=1}^n \frac{y_i}{z_i} \right )^{\frac{1}{n}} \leq \frac{1}{n} \left ( \sum_{i=1}^n \frac{y_i}{z_i} \right ). \]

  6. Multiplying the first inequality by \(t\) and the second by \((1-t)\) and adding them, we get:

    \[ \frac{t f(\bx) + (1-t) f(\by)}{f(\bz)} \leq \frac{1}{n} \left ( \sum_{i=1}^n\frac{t x_i}{z_i} + \frac{(1-t) y_i}{z_i} \right ). \]

  7. Recall that \(z_i = t x_i + (1-t) y_i\). Thus, the inequality above simplifies to:

    \[\begin{split} &\frac{t f(\bx) + (1-t) f(\by)}{f(\bz)} \leq 1 \\ &\iff t f(\bx) + (1-t) f(\by) \leq f(\bz) = f(t \bx + (1-t) \by). \end{split}\]

  8. Thus, \(f\) is concave.

Example 6.29 (Powers of absolute value)

Let \(f : \RR \to \RR\) be:

\[ f(x) = |x|^p \]

with \(\dom f = \RR\).

For \(p=1\), the absolute value function \(f(x) = |x|\) is convex.

Consider the case where \(p > 1\). Let \(q\) be its conjugate exponent given by \(\frac{1}{p} + \frac{1}{q} = 1\).

Let \(x, y \in \RR\) and \(s, t \in (0, 1)\) with \(s + t= 1\).

  1. By triangle inequality:

    \[ | s x + t y | \leq s | x| + t |y|. \]

  2. We can write this as:

    \[ s | x| + t |y| = \left ( s^{\frac{1}{p}} |x| \right ) s^{\frac{1}{q}} + \left ( t^{\frac{1}{p}} |y| \right ) t^{\frac{1}{q}}. \]

  3. By Hölder's inequality, for some \(a_1, a_2, b_1, b_2 \geq 0\)

    \[ a_1 b_1 + a_2 b_2 \leq (a_1^p + a_2^p)^{\frac{1}{p}} + (b_1^q + b_2^q)^{\frac{1}{q}}. \]

  4. Let \(a_1 = s^{\frac{1}{p}} |x|\), \(a_2 = t^{\frac{1}{p}} |y|\), \(b_1 = s^{\frac{1}{q}}\) and \(b_2 = t^{\frac{1}{q}}\).

  5. Applying Hölder’s inequality,

    \[ | s x + t y | \leq s | x| + t |y| \leq (s |x|^p + t |y|^p)^{\frac{1}{p}} (s + t)^{\frac{1}{q}} = (s |x|^p + t |y|^p)^{\frac{1}{p}}. \]

  6. Taking power of \(p\) on both sides, we get:

    \[ | s x + t y |^p \leq s |x|^p + t |y|^p. \]

  7. Which is the same as

    \[ f(s x + t y) \leq s f(x) + t f(y). \]

  8. Thus, \(f\) is convex.

Observation 6.4 (Empty function is convex)

Let \(f: \VV \to \RR\) be such that \(\dom f = \EmptySet\). Then \(f\) is convex.

Proof. The empty set \(\dom f\) is vacuously convex. The defining inequality (6.1) is vacuously true for \(f\) since its domain is empty.

This observation may sound uninteresting however it is important in the algebra of convex functions. Definition 2.48 provides definitions for sum of two partial functions and scalar multiplication with a partial function. If two convex functions \(f\) and \(g\) are such that \(\dom f \cap \dom g = \EmptySet\), then their sum \(f+g\) is empty function. The observation above says that \(f+g\) is still a convex function.

Jensen's inequality, discussed later in the section, generalizes the notion of convexity of a function to arbitrary convex combinations of points in its domain.

6.6.2. Convexity on Lines in Domain¶

Theorem 6.67 (\(f\) is convex = \(f\) is convex on lines in domain)

Let \(\VV\) be a real vector space. A function \(f : \VV \to \RR\) is convex if and only if for any \(\bx \in \dom f\) and any \(\bv \in \VV\), the function \(g(t) = f(\bx + t\bv)\) is convex (on its domain , \(\{ t \in \RR \ST \bx + t\bv \in \dom f\}\)).

In other words, \(f\) is convex if and only if it is convex when restricted to any line that intersects its domain.

Proof. For any arbitrary \(f\), and for any \(\bx \in \dom f, \bv \in \VV\)
we have defined \(g : \RR \to \RR\) as:

\[ g(t) \triangleq f(\bx + t\bv) \]

with the domain:

\[ \dom g = \{t \ST \bx + t \bv \in \dom f\}. \]

Assume \(f\) to be convex. To show that \(g\) is convex, we need to show that \(\dom g\) is convex and \(g\) satisfies the convexity inequality.

  1. Let \(t_1, t_2 \in \dom g\).

  2. It means that there are \(\by = \bx + t_1 \bv \in \dom f\) and \(\bz = \bx + t_2 \bv \in \dom f\),

  3. and \(g(t_1) = f(\by)\) and \(g(t_2) = f(\bz)\).

  4. Let \(r \in [0,1]\) and \(t = rt_1 + (1-r)t_2\).

  5. Note that:

    \[\begin{split} \bx + t \bv &= \bx + (rt_1 + (1-r)t_2) \bv\\ &= r\bx + (1-r)\bx + rt_1 \bv + (1-r)t_2 \bv\\ &= r (\bx + t_1 \bv) + (1-r)(\bx + t_2 \bv)\\ &= r \by + (1-r) \bz. \end{split}\]

  6. Since \(\dom f\) is convex, hence \(r \by + (1-r) \bz \in \dom f\).

  7. Thus, \(g\) is defined at \(t = rt_1 + (1-r)t_2\) for all \(r \in [0,1]\).

  8. We have shown that if \(t_1, t_2 \in \dom g\), then \(t = rt_1 + (1-r)t_2 \in \dom g\) for all \(r \in [0,1]\).

  9. Thus, \(\dom g\) is convex.

  10. Now,

    \[\begin{split} g(t) &= g(r t_1 + (1-r)t_2)\\ &= f(r \by + (1-r) \bz)\\ &\leq r f(\by) + (1-r) f(\bz)\\ &= r g(t_1) + (1-r) g(t_2). \end{split}\]

  11. We showed that for any \(t_1, t_2 \in \dom g\) and \(r \in [0,1]\),

    \[ g (r t_1 + (1-r) t_2 )\leq r g(t_1) + (1-r) g(t_2). \]

  12. Thus, \(g\) is convex.

For the converse, we need to show that if for any \(\bx \in \dom f\) and any \(\bv \in \VV\), \(g\) is convex, then \(f\) is convex.

Assume for contradiction that \(f\) is not convex. Then, either \(\dom f\) is not convex or there exists some \(\bx, \by \in \dom f\) and \(t \in [0,1]\) such that:

\[ f((1-t)\bx + t\by) > (1 - t) f(\bx) + t f(\by). \]

We show that both of these conditions lead to contradictions.

  1. Assume \(\dom f\) is not convex.

  2. Then, there exists \(\bx, \by \in \dom f\) such that for some \(t \in [0,1]\):

    \[ (1 - t)\bx + t \by \notin \dom f. \]

  3. Let \(\bv = \by - \bx\).

  4. Note that \((1 - t)\bx + t \by = \bx + t \bv\).

  5. Picking \(g\) for this particular choice of \(\bx\) and \(\bv\), we note that \(g\) is defined at \(t=0\) and \(t=1\) since they corresponding to the points \(\bx\) and \(\by\) respectively in \(\dom f\).

  6. Since \(g\) is convex by hypothesis, hence, \(g\) is defined at every \(t \in [0,1]\).

  7. This contradicts the fact that \(g\) is not defined at some \(t \in [0,1]\).

  8. Thus, \(\dom f\) must be convex.

We now accept that \(\dom f\) is convex, but there exists some \(\bx, \by \in \dom f\) and \(t \in [0,1]\) such that:

\[ f((1-t)\bx + t\by) > (1 - t) f(\bx) + t f(\by). \]

  1. Again, let \(\bv = \by - \bx\) and note that \((1 - t)\bx + t \by = \bx + t \bv\).

  2. Pick \(g\) for this choice of \(\bx\) and \(\bv\).

  3. We have \(g(t) = f(\bx + t \bv)\).

  4. In particular, \(g(0) = f(\bx)\) and \(g(1) = f(\by)\).

  5. Since \(g\) is convex, hence for every \(t \in [0,1]\),

    \[\begin{split} f((1-t)\bx + t\by) &= f(\bx + t \bv)\\ &= g(t)\\ &= g ((1-t)0 + t 1)\\ &\leq (1-t)g(0) + t g(1)\\ &= (1-t)f(\bx) + t f(\by). \end{split}\]

  6. We have a contradiction.

  7. Thus, \(f\) must be convex.

A good application of this result is in showing the concavity of the log determinant function in Example 6.45 below.

6.6.3. Epigraph¶

The epigraph of a function \(f: \VV \to \RR\) is given by:

\[ \epi f = \{ (\bx,t) \in \VV \oplus \RR \ST \bx \in \dom f, f(\bx) \leq t \}. \]

\(\VV \oplus \RR\) is the direct sum of \(\VV\) and \(\RR\) having appropriate vector space structure.

6.6.3.1. Convex Functions¶

Theorem 6.68 (Function convexity = Epigraph convexity)

Let \(\VV\) be a real vector space. A function \(f: \VV \to \RR\) is convex if and only if its epigraph \(\epi f\) is a convex set.

Proof. Let \(C = \epi f\).

Assume \(f\) is convex.

  1. Let \((\bx_1, t_1), (\bx_2, t_2) \in \epi f\).

  2. Then, \(f(\bx_1) \leq t_1\) and \(f(\bx_2) \leq t_2\).

  3. Let \(r \in [0,1]\).

  4. Consider the point \((\bx, t) = r(\bx_1, t_1) + (1-r)(\bx_2, t_2)\).

  5. We have \(\bx = r \bx_1 + (1 -r)(\bx_2)\).

  6. And \(t = r t_1 + (1-r)t_2\).

  7. Then, \( r f(\bx_1) + (1-r) f(\bx_2) \leq r t_1 + (1-r)t_2 = t\).

  8. Since \(f\) is convex, hence \(f(\bx) = f(r \bx_1 + (1 -r)(\bx_2)) \leq r f(\bx_1) + (1-r)f(\bx_2)\).

  9. But then, \(f(\bx) \leq t\).

  10. Thus, \((\bx, t) \in \epi f\).

  11. Thus, \((\bx_1, t_1), (\bx_2, t_2) \in \epi f\) implies \(r(\bx_1, t_1) + (1-r)(\bx_2, t_2) \in \epi f\) for all \(r \in [0,1]\).

  12. Thus, \(\epi f\) is convex.

Assume \(\epi f\) is convex.

  1. Let \(\bx_1, \bx_2 \in \dom f\).

  2. Let \(t_1 = f(\bx_1)\) and \(t_2 = f(\bx_2)\).

  3. Then, \((\bx_1, t_1), (\bx_2, t_2) \in \epi f\).

  4. Let \(r \in [0,1]\).

  5. Let \(\bx = r \bx_1 + (1-r)\bx_2\).

  6. Since \(\epi f\) is convex, hence \(r(\bx_1, t_1) + (1-r)(\bx_2, t_2) \in \epi f\).

  7. i.e., \((r \bx_1 + (1-r)\bx_2, r t_1 + (1-r)t_2) \in \epi f\).

  8. Thus, \(\bx = r \bx_1 + (1-r)\bx_2 \in \dom f\).

  9. Thus, \(\dom f\) is convex.

  10. And, \(f(\bx) \leq r t_1 + (1-r)t_2\).

  11. But \(r t_1 + (1-r)t_2 = r f(\bx_1) + (1-r)f(\bx_2)\).

  12. Thus, \(f(r \bx_1 + (1-r)\bx_2) \leq r f(\bx_1) + (1-r)f(\bx_2)\).

  13. Thus, \(f\) is convex as \(\bx_1, \bx_2 \in \dom f\) and \(r \in [0,1]\) were arbitrary.

Note that \(\dom f\) is the projection of \(\epi f\) from \(\VV \oplus \RR\) to \(\VV\). Due to Theorem 6.30, \(\epi f\) convex implies \(\dom f\) convex as the projection is a linear operation.

6.6.3.2. Nonnegative Homogeneous Functions¶

Recall that a real valued function \(f : \VV \to \RR\) is nonnegative homogeneous if for every \(t \in \RR_+\), \(f(t \bx) = t f(\bx)\).

Theorem 6.69 (Nonnegative homogeneity = Epigraph is cone)

A function \(f : \VV \to \RR\) is nonnegative homogeneous if and only if its epigraph \(\epi f\) is a cone in \(\VV \oplus \RR\).

Proof. Let \(f\) be nonnegative homogeneous.

  1. Let \((\bx, f(\bx)) \in \epi f\).

  2. Let \(t \geq 0\).

  3. Then, \(t(\bx, f(\bx)) = (t\bx, t f(\bx)) = (t\bx, f(t\bx))\).

  4. But \((t\bx, f(t\bx)) \in \epi f\) since \(f\) is nonnegative homogeneous.

  5. Thus, \(\VV \oplus \RR\) is closed under nonnegative scalar multiplication.

  6. Thus, \(\epi f\) is a cone.

Assume \(\epi f\) is a cone.

  1. Let \(\bx \in \dom f\).

  2. Then, \((\bx, f(\bx)) \in \epi f\).

  3. Since \((\bzero, 0) \in \epi f\) as \(\epi f\) is a cone, hence \(f(0 \bx) = 0 f(\bx) = 0\).

  4. Now, let \(t > 0\).

  5. Since \(\epi f\) is a cone then, \(t(\bx, f(\bx)) \in \epi f\).

  6. Then, \(t(\bx, f(\bx)) = (t\bx, t f(\bx)) \in \epi f\).

  7. By definition of epigraph, \(f(t\bx) \leq t f(\bx)\).

  8. We claim that \(f(t \bx) = t f(\bx)\) must hold true.

  9. Assume for contradiction that \(f(t \bx) = s f(\bx)\) where \(s < t\).

  10. Then, \((t\bx, s f(\bx)) \in \epi f\).

  11. Since \(\epi f\) is a cone, hence dividing by \(t\), we get, \((\bx, \frac{s}{t} f(\bx)) \in \epi f\).

  12. But then, \(f(\bx) \leq \frac{s}{t} f(\bx) < f(\bx)\) which is a contradiction.

  13. Hence, \(f(t \bx) = t f(\bx)\) must hold true.

6.6.3.3. Nonnegative Homogeneous Convex Functions¶

Theorem 6.70 (Nonnegative homogeneous convex function epigraph characterization)

Let \(\VV\) be a real vector space. A real valued function \(f: \VV \to \RR\) is nonnegative homogeneous and convex if and only if its epigraph \(\epi f\) is a convex cone.

Proof. By Theorem 6.69, \(f\) is nonnegative homogeneous if and only if \(\epi f\) is a cone.

By Theorem 6.68, \(f\) is convex if and only if \(\epi f\) is convex.

Thus, \(f\) is nonnegative homogeneous and convex if and only if its epigraph \(\epi f\) is a convex cone.

Theorem 6.71 (Nonnegative homogeneous convex function is subadditive)

Let \(\VV\) be a real vector space. A nonnegative homogeneous function \(f: \VV \to \RR\) is convex if and only if it is subadditive:

\[ f(\bx + \by) \leq f(\bx) + f(\by) \Forall \bx, \by \in \dom f. \]

Proof. Assume that \(f\) is nonnegative homogeneous and convex.

  1. By Theorem 6.70, \(\epi f\) is a convex cone.

  2. Then, by Theorem 6.44 \(\epi f\) is closed under addition and nonnegative scalar multiplication.

  3. Pick any \(\bx, \by \in \dom f\).

  4. Then \((\bx, f(\bx)), (\by, f(\by)) \in \epi f\).

  5. Then, their sum \((\bx + \by, f(\bx) + f(\by)) \in \epi f\).

  6. This means that \(f(\bx + \by) \leq f(\bx) + f(\by)\) by definition of epigraph.

Now for the converse, assume that \(f\) is nonnegative homogeneous and subadditive.

  1. By Theorem 6.69, \(\epi f\) is a cone.

  2. Consequently, it is closed under nonnegative scalar multiplication.

  3. Pick any \(\bx, \by \in \dom f\).

  4. Let \((\bx, \alpha), (\by, \beta) \in \epi f\).

  5. Then, \(f(\bx) \leq \alpha\) and \(f(\by) \leq \beta\).

  6. Now, \((\bx + \by, f(\bx + \by)) \in \epi f\).

  7. Since \(f\) is subadditive, hence \(f(\bx + \by) \leq f(\bx) + f(\by) \leq \alpha + \beta\).

  8. Thus, \((\bx + \by, \alpha + \beta) \in \epi f\).

  9. We have shown that for any \((\bx, \alpha), (\by, \beta) \in \epi f\), \((\bx + \by, \alpha + \beta) \in \epi f\).

  10. Thus, \(\epi f\) is closed under vector addition.

  11. Since \(\epi f\) is closed under vector addition and nonnegative scalar multiplication, hence \(\epi f\) is a convex cone.

  12. But then, by Theorem 6.70, \(f\) is convex.

Corollary 6.5

Let \(\VV\) be a real vector space. Let \(f: \VV \to \RR\) be a nonnegative homogeneous convex function. Then,

\[ f(-\bx) \geq -f(\bx) \Forall \bx \in \dom f. \]

Proof. Since \(f\) is nonnegative homogeneous convex, hence \(f\) is subadditive.

Thus,

\[ f(\bx - \bx) \leq f(\bx) + f(-\bx). \]

But,

\[ f(\bx - \bx) = f(\bzero) = f(0 \bzero) = 0 f(\bzero) = 0. \]

Thus,

\[ f(\bx) + f(-\bx) \geq 0. \]

Thus,

\[ f(-\bx) \geq -f(\bx). \]

Theorem 6.72 (Linearity of nonnegative homogeneous functions)

Let \(\VV\) be a real vector space. Let \(f: \VV \to \RR\) be a nonnegative homogeneous convex function. Then, \(f\) is linear on a subspace \(L\) of \(\VV\) if and only if \(f(-\bx) = -f(\bx)\) for every \(\bx \in L\).

If \(L\) is finite dimensional, then this is true if merely \(f(-\bb_i) = -f(\bb_i)\) for all the vectors in some basis \(\bb_1, \dots, \bb_n\) for \(L\).

Proof. We are given that \(f\) is nonnegative homogeneous convex.

Assume that \(f\) is linear over \(L\). Then by definition of linearity, \(f(-\bx) = -f(\bx)\) for every \(\bx \in L\).

Now, for the converse, assume that \(f(-\bx) = -f(\bx) \Forall \bx \in L\).

  1. Let \(\bx, \by \in L\).

  2. Then, \(f(\bx + \by) \leq f(\bx) + f(\by)\) since \(f\) is subadditive.

  3. Also,

    \[ f(\bx + \by) = -f(-(\bx + \by)) = -f(-\bx + (-\by)). \]

  4. But

    \[ f(-\bx + (-\by)) \leq f(-\bx) + f(-\by) = -f(\bx) - f(\by) = - (f(\bx) + f(\by)). \]

  5. Thus,

    \[ f(\bx + \by) = -f(-\bx + (-\by)) \geq f(\bx) + f(\by). \]

  6. Combining, we get \(f(\bx + \by) = f(\bx) + f(\by)\). Thus, \(f\) is additive over \(L\).

  7. For any \(t < 0\),

    \[ f(t \bx) = - f(-t \bx) = - (-t) f(\bx) = t f(\bx). \]

  8. Thus, for any \(t \in \RR\), \(f(t \bx) = t f(\bx)\).

  9. Thus, \(f\) is homogeneous over \(L\).

  10. Since \(f\) is additive and homogeneous over \(L\), hence \(f\) is linear.

Finally, assume that \(L\) is finite dimensional and has a basis \(\bb_1, \dots, \bb_n\).

  1. By previous argument \(f\) is homogeneous on the basis vectors; i.e., \(f(t \bb_i) = t f(\bb_i)\) for any \(t \in \RR\) and any basis vector.

  2. Let \(\bx \in L\).

  3. Then, \(\bx = \sum_{i=1}^n a_i \bb_i\).

  4. Since \(f\) is subadditive, hence

    \[\begin{split} f(a_1 \bb_1) + \dots + f(a_n \bb_n) & \geq f(a_1 \bb_1 + \dots + a_n \bb_n)\\ &= f(\bx) \\ &\geq - f(-\bx)\\ &= - f(-a_1 \bb_1 - \dots - a_n \bb_n)\\ &\geq - f(-a_1 \bb_1) - \dots - f(-a_n \bb_n)\\ &=f(a_1 \bb_1) + \dots + f(a_n \bb_n). \end{split}\]

  5. This can hold only if all the inequalities are equalities in the previous derivation.

  6. Thus, \(f(\bx) = - f(-\bx)\) or \(-f(\bx) = f(-\bx)\) for every \(\bx \in L\).

  7. Then, following the previous argument, \(f\) is linear over \(L\).

6.6.4. Extended Value Extensions¶

Tracking domains of convex functions is difficult. It is often convenient to extend a convex function \(f : \VV \to \RR\) with a domain \(\dom f \subset \VV\) to all of \(\VV\) by defining it to be \(\infty\) outside its domain.

Definition 6.44 (Extended value extension)

The extended value extension of a convex function \(f: \VV \to \RR\) is defined as \(\tilde{f} : \VV \to \ERL\) given by:

\[\begin{split} \tilde{f}(\bx) \triangleq \begin{cases} f(\bx) & \text{for} & \bx \in \dom f \\ \infty & \text{for} & \bx \notin \dom f. \end{cases} \end{split}\]

We mention that the defining inequality (6.1) for a convex function \(f\) remains valid (rather gets extended) for its extended value extension too. If \(f\) is convex over \(\dom f\), then for any \(\bx_1, \bx_2 \in \VV\), and \(0 < t < 1\), we have

\[ \tilde{f}(t \bx_1 + (1-t) \bx_2) \leq t \tilde{f}(\bx_1) + (1-t) \tilde{f}(\bx_2). \]

  1. At \(t=0\) and \(t=1\), this is always an equality.

  2. Since \(\dom f\) is convex, hence if both \(\bx_1, \bx_2 \in \dom f\), then \(t \bx_1 + (1-t) \bx_2 \in \dom f\) too.

  3. The inequality then reduces to (6.1).

  4. If either \(\bx_1\) or \(\bx_2\) is not in \(\dom f\), then the R.H.S. becomes \(\infty\) and the inequality stays valid.

Definition 6.45 (Effective domain)

The effective domain of an extended valued function \(\tilde{f} : \VV \to \ERL\) is defined as:

\[ \dom \tilde{f} = \{ \bx \in \VV \ST \tilde{f}(\bx) < \infty \}. \]

An equivalent way to define the effective domain is:

\[ \dom \tilde{f} = \{\bx \in \VV \ST \exists t \in \RR \text{ with } (\bx, t) \in \epi f \}. \]

Theorem 6.73 (Effective domain and sum of functions)

If \(f\) and \(g\) are two extended valued functions, then \(\dom (f + g) = \dom f \cap \dom g\).

Proof. We proceed as follows:

  1. At any \(\bx \in \dom f \cap \dom g\), both \(f(\bx)\) and \(g(\bx)\) are finite.

  2. Thus, \(f(\bx) + g(\bx)\) is finite.

  3. Hence, \(\bx \in \dom (f + g)\).

  4. At any \(\bx \in \VV \setminus (\dom f \cap \dom g)\), either \(f(\bx) = \infty\) or \(g(\bx) = \infty\).

  5. Thus, \((f + g)(\bx) = f(\bx) + g(\bx) = \infty\).

  6. Thus, \(\bx \notin \dom (f + g)\).

Several commonly used convex functions have vastly different domains. If we work with these functions directly, then we have to constantly worry about identifying the domain and manipulating the domain as per the requirements of the operations we are considering. This quickly becomes tedious.

An alternative approach is to work with the extended value extensions of all functions. We don’t have to worry about tracking the function domain. The domain can be identified whenever needed by removing the parts from \(\VV\) where the function goes to \(\infty\).

In this book, unless otherwise specified, we shall assume that the functions are being treated as their extended value extensions.

6.6.5. Proper Functions¶

Definition 6.46 (Proper function)

An extended real-valued function \(f : \VV \to \ERL\) is called proper if its domain is nonempty, it never takes the value \(-\infty\) and is not identically equal to \(\infty\).

\[ \exists \bx \in \VV \text{ such that } f(\bx) < \infty \text{ and } f(\bx) > -\infty \Forall \bx \in \VV. \]

In other words, \(\epi f\) is nonempty and contains no vertical lines.

Putting another way, a proper function is obtained by taking a real valued function \(f\) defined on a nonempty set \(C \subseteq \VV\) and then extending it to all of \(\VV\) by setting \(f(\bx) = +\infty\) for all \(\bx \notin C\).

It is easy to see that the codomain for a proper function can be changed from \(\ERL\) to \(\RERL\) to clarify that it never takes the value \(-\infty\).

Definition 6.47 (Improper function)

An extended real-valued function \(f : \VV \to \ERL\) is called improper if it is not proper.

For an improper function \(f\):

  • \(\dom f\) may be empty.

  • \(f\) might take a value of \(-\infty\) at some \(\bx \in \VV\).

Most of our study is focused on proper functions. However, improper functions sometimes do arise naturally in convex analysis.

Example 6.30 (An improper function)

Consider a function \(f: \RR \to \ERL\) as described below:

\[\begin{split} f(x) = \begin{cases} -\infty & \text{ if } & | x | < 1 \\ 0 & \text{ if } & |x| = 1 \\ \infty & \text{ if } & | x | > 1 . \end{cases} \end{split}\]

Then, \(f\) is an improper function.

6.6.6. Indicator Functions¶

Definition 6.48 (Indicator function)

Let \(C \subseteq \VV\). Then, its indicator function is given by \(I_C(\bx) = 0 \Forall \bx \in C\). Here, \(\dom I_C = C\).

The extended value extension of an indicator function is given by:

\[\begin{split} \tilde{I_C}(\bx) \triangleq \begin{cases} 0 & \text{for} & \bx \in C \\ \infty & \text{for} & \bx \notin C. \end{cases} \end{split}\]

Theorem 6.74 (Indicator functions and convexity)

An indicator function is convex if and only if its domain is a convex set.

Proof. Let \(C\) be convex. Let \(I_C\) be its indicator function. Let \(\bx, \by \in C\). Then, for any \(t \in [0,1]\)

\[ I_C(t \bx + (1-t) \by) = 0 \leq t I_C(\bx) + (1-t) I_C(\bx) = 0. \]

since \(t \bx + (1-t) \by \in C\) as \(C\) is convex.

Thus, \(I_C\) is convex.

If \(C\) is not convex, then \(\dom I_C\) is not convex. Hence, \(I_C\) is not convex.

Theorem 6.75 (Restricting the domain of a function)

Let \(f\) be a proper function. Then,

\[ \dom (f + I_C) = \dom f \cap C. \]

Also,

\[ (f + I_C )(\bx) = f(\bx) \Forall \bx \in \dom f \cap C. \]

The statement is obvious. And quite powerful.

  • The problem of minimizing a function \(f\) over a set \(C\) is same as minimizing \(f + I_C\) over \(\VV\).

6.6.7. Sublevel Sets¶

Recall from Definition 2.52 that the \(\alpha\)-sublevel set for a real valued function \(f : \VV \to \RR\) is given by

\[ C_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) \leq \alpha \}. \]

The open sublevel sets for a real valued function \(f: \VV \to \RR\) can be defined as

\[ O_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) < \alpha \}. \]

The closed sublevel sets can be shown to be intersection of a set of open sublevel sets.

Theorem 6.76 (Sublevel set as intersection)

Let \(\VV\) be a real vector space. Let \(f : \VV \to \RR\) be a real valued function. Let

\[ O_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) < \alpha \} \]

denote the open sublevel set of \(f\) for \(\alpha\).

Let

\[ C_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) \leq \alpha \} \]

denote the closed sublevel set of \(f\) for \(\alpha\). Then,

\[ C_{\alpha} = \bigcap_{\mu > \alpha} O_{\mu}. \]

Proof. We show that \(C_{\alpha} \subseteq \bigcap_{\mu > \alpha} O_{\mu}\).

  1. Let \(\bx \in C_{\alpha}\).

  2. Then, \(f(\bx) \leq \alpha\).

  3. Thus, \(f(\bx) < \mu\) for every \(\mu > \alpha\).

  4. Thus, \(\bx \in O_{\mu}\) for every \(\mu > \alpha\).

  5. Thus, \(C_{\alpha} \subseteq \bigcap_{\mu > \alpha} O_{\mu}\).

We now show that \(C_{\alpha} \supseteq \bigcap_{\mu > \alpha} O_{\mu}\).

  1. Let \(\bx \in \bigcap_{\mu > \alpha} O_{\mu}\).

  2. Then, \(f(\bx) < \mu\) for every \(\mu > \alpha\).

  3. Taking the infimum on the R.H.S. over the set \(\{ \mu \in \RR \ST \mu > \alpha \}\), we obtain

    \[ f(\bx) \leq \inf \{ \mu \in \RR \ST \mu > \alpha \} = \alpha. \]

  4. Thus, \(\bx \in C_{\alpha}\).

  5. Thus, \(\bigcap_{\mu > \alpha} O_{\mu} \subseteq C_{\alpha}\).

Theorem 6.77 (Convexity of sublevel sets)

If \(f : \VV \to \RR\) is convex, then its sublevel sets given by

\[ C_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) \leq \alpha \} \]

are convex.

Proof. Assume \(f\) is convex.

  1. Let \(\bx, \by \in C_{\alpha}\).

  2. Then, \(f(\bx) \leq \alpha\) and \(f(\by) \leq \alpha\).

  3. Let \(t \in [0,1]\).

  4. Let \(\bz = t \bx + (1-t)\by\).

  5. Since \(f\) is convex, hence:

    \[ f(\bz) \leq t f(\bx) + (1-t) f(\by) \leq t \alpha + (1-t) \alpha = \alpha. \]

  6. Thus, \(f(\bz) \leq \alpha\).

  7. Thus, \(\bz \in C_{\alpha}\).

  8. Thus, \(C_{\alpha}\) is convex.

The converse is not true. A function need not be convex even if all its sublevel sets are convex.

Example 6.31

Consider the function \(f(x) = \ln x\). It is concave (Example 6.41).

Its sublevel sets are convex as they are intervals.

Theorem 6.78 (Convexity of open sublevel sets)

If \(f : \VV \to \RR\) is convex, then its open sublevel sets given by

\[ O_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) < \alpha \} \]

are convex.

Proof. Assume \(f\) is convex.

  1. Let \(\bx, \by \in O_{\alpha}\).

  2. Then, \(f(\bx) < \alpha\) and \(f(\by) < \alpha\).

  3. Let \(t \in (0,1)\).

  4. Let \(\bz = t \bx + (1-t)\by\).

  5. Since \(f\) is convex, hence:

    \[ f(\bz) \leq t f(\bx) + (1-t) f(\by) < t \alpha + (1-t) \alpha = \alpha. \]

  6. Thus, \(f(\bz) < \alpha\).

  7. Thus, \(\bz \in O_{\alpha}\).

  8. Thus, \(O_{\alpha}\) is convex.

An alternate proof for showing the convexity of the closed sublevel sets is to show it as an intersection of open sublevel sets.

Theorem 6.79 (Intersection of sublevel sets of convex functions)

Let \(\VV\) be a real vector space. Let \(I\) be an arbitrary index set. Let \(f_i : \VV \to \RERL\) be convex functions for every \(i \in I\). Let \(\alpha_i \in \RR\) for every \(i \in I\). Then,

\[ C = \{ \bx \ST f_i(\bx) \leq \alpha_i, \Forall i \in I \} \]

is a convex set.

Proof. For each \(i \in I\), consider the set

\[ C_i = \{ \bx \ST f_i(\bx) \leq \alpha_i\}. \]

Then, \(C_i\) is a sublevel set of \(f_i\) which is convex. Hence, \(C_i\) is convex. Now, we can see that

\[ C = \bigcap_{i \in I}C_i \]

Thus, \(C\) is an intersection of convex sets. Hence, \(C\) is convex.

Example 6.32 (Convexity of sublevel sets of the quadratic)

Let \(\bQ \in \SS^n\) be a positive semidefinite matrix. Let \(\ba \in \RR^n\) and \(c \in \RR\).

Consider the sets of the form

\[ \{\bx \in \RR^n \ST \frac{1}{2} \langle \bx, \bQ \bx \rangle + \langle \bx, \ba \rangle + c \leq 0 \}. \]

This is a sublevel set of the quadratic function \(f(\bx) = \frac{1}{2} \langle \bx, \bQ \bx \rangle + \langle \bx, \ba \rangle + c\).

Since \(f\) is convex, hence the set is convex.

Sets of this form include the solid ellipsoids, paraboloids as well as spherical balls. Here is an example of the spherical ball of the norm induced by the inner product.

\[ \{\bx \ST \| \bx \| \leq 1 \} = \{ \bx \ST \langle \bx, \bx \rangle - 1 \leq 0 \}. \]

6.6.8. Hypograph¶

The hypograph of a function \(f: \VV \to \RR\) is given by:

\[ \hypo f = \{ (\bx,t) \in \VV \oplus \RR \ST \bx \in \dom f, f(\bx) \geq t \}. \]

Just like function convexity is connected to epigraph convexity, similarly function concavity is connected to hypograph convexity.

Theorem 6.80 (Function concavity = Epigraph convexity)

A function \(f\) is concave if and only if its hypograph \(\hypo f\) is a convex set.

Proof. \(f\) is concave if and only if \(-f\) is convex if and only if the epigraph of \(-f\) is convex if and only if the hypograph of \(f\) is convex.

6.6.9. Super-level Sets¶

Recall from Definition 2.54 that the \(\alpha\)-sublevel set for a real valued function \(f : \VV \to \RR\) is given by

\[ D_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) \geq \alpha \}. \]

Theorem 6.81

If \(f : \VV \to \RR\) is concave, then its super-level sets are convex.

Proof. Assume \(f\) is concave.

  1. Let \(\bx, \by \in D_{\alpha}\).

  2. Then, \(f(\bx) \geq \alpha\) and \(f(\by) \geq \alpha\).

  3. Let \(t \in [0,1]\).

  4. Let \(\bz = t \bx + (1-t)\by\).

  5. Since \(f\) is concave, hence:

    \[ f(\bz) \geq t f(\bx) + (1-t) f(\by) \geq t \alpha + (1-t) \alpha = \alpha. \]

  6. Thus, \(f(\bz) \geq \alpha\).

  7. Thus, \(\bz \in D_{\alpha}\).

  8. Thus, \(D_{\alpha}\) is convex.

The converse is not true. A function need not be concave even if all its super-level sets are convex.

Example 6.33

Let geometric mean be given by:

\[ g(\bx) = \left ( \prod_{i=1}^n x_i \right )^{\frac{1}{n}} \]

with \(\dom g = \RR^n_{+}\) (with the interpretation that \(0^{\frac{1}{n}} = 0\)).

Let arithmetic mean be given by:

\[ a(\bx) = \frac{1}{n}\sum_{i=1}^n x_i. \]

Consider the set:

\[ A = \{ \bx \in \RR^n_+ \ST g(\bx) \geq \alpha a(\bx) \}. \]

\(A\) is the set of nonnegative vectors where the geometric mean of the components is at least \(\alpha\) times larger than the arithmetic mean.

  1. In Example 6.28 we establish that \(g\) is concave.

  2. In Example 6.24 we established that \(a\) is convex as well as concave.

  3. Thus, \(-\alpha a\) is concave.

  4. Thus, \(g - \alpha a\) is concave (with \(\dom g - \alpha a = \RR^n_+\)).

  5. Note that \(A\) can be redefined as:

    \[ A = \{ \bx \in \RR^n_+ \ST g(\bx) - \alpha a(\bx) \geq 0 \}. \]

  6. Thus, \(A\) is a \(0\)-super-level set of \(g - \alpha a\).

  7. Since \(g - \alpha a\) is concave, hence \(A\) is convex.

6.6.10. Support Functions¶

Definition 6.49 (Support function for a set)

Let \(\VV\) be a real inner product space. Let \(C\) be a subset of \(\VV\). The support function \(\sigma_C : \VV \to \RERL\) is defined as

\[ \sigma_C (\bx) = \sup \{\langle \bx, \by \rangle \ST \by \in C \}. \]

Since \(\VV\) and \(\VV^*\) are isomorphic, support function \(\sigma_C : \VV^* \to \RERL\) for a set \(C \subseteq \VV^*\) is similarly defined as

\[ \sigma_C (\bx) = \sup \{\langle \by, \bx \rangle \ST \by \in C \}. \]

Theorem 6.82 (Convexity of support function)

Let \(\VV\) be a real inner product space. Let \(C\) be a nonempty subset of \(\VV\). Then, the support function \(\sigma_C : \VV \to \RERL\) is convex.

Proof. Fix a \(\by \in C\) and consider the function \(\sigma_{\by} : \VV \to \RR\) given by

\[ \sigma_{\by} (\bx) = \langle \bx, \by \rangle. \]

\(\sigma_{\by}\) is linear and accordingly convex. Then,

\[ \sigma_C (\bx) = \sup_{\by \in C} \sigma_{\by} (\bx) \]

is a pointwise supremum of convex functions. By Theorem 6.109, \(\sigma_C\) is convex.

We note that the convexity of the support function \(\sigma_C\) has nothing to do with the convexity of the underlying set \(C\).

Theorem 6.83 (Closedness of support function)

Let \(\VV\) be a real inner product space. Let \(C\) be a nonempty subset of \(\VV\). Then, the support function \(\sigma_C : \VV \to \RERL\) is closed.

Proof. Recall that a function is closed if all its sublevel sets are closed.

  1. Let \(a \in \RR\).

  2. Consider the sublevel set \(S_a = \{ \bx \in \VV \ST \sigma_C(\bx) \leq a \}\).

  3. Then,

    \[ S_a = \{ \bx \in \VV \ST \sup_{\by \in C} \langle \bx, \by \rangle \leq a \} \]

  4. Thus,

    \[ S_a = \{ \bx \in \VV \ST \langle \bx, \by \rangle \leq a \Forall \by \in C \} \]

  5. Define \(A_y\) as

    \[ A_y = \{ \bx \in \VV \ST \langle \bx, \by \rangle \leq a \}. \]

  6. Then,

    \[ S_a = \bigcap_{\by \in C} A_y. \]

  7. Now, \(A_y\) is a closed set since \(\langle \bx, \by \rangle\) is a continuous function.

  8. Thus, \(S_a\) is an intersection of closed sets.

  9. Thus, \(S_a\) is closed.

  10. Thus, all sublevel sets of \(\sigma_C\) are closed.

  11. Thus, \(\sigma_C\) is closed.

Theorem 6.84 (Equality of underlying sets for support functions)

Let \(A, B \subseteq \VV\) be nonempty, closed and convex sets. Then, \(A = B\) if and only if \(\sigma_A = \sigma_B\).

Proof. If \(A = B\) then obviously \(\sigma_A = \sigma_B\). Now, assume that \(\sigma_A = \sigma_B\).

  1. For contradiction, assume that \(A \neq B\).

  2. Without loss of generality, assume that there exists \(\by \in A\) such that \(\by \notin B\).

  3. Since \(\by \notin B\) and \(B\) is a closed convex set, hence, by Theorem 6.153, there exists a hyperplane \(H\) strongly separating \(\by\) from \(B\).

  4. Thus, there exists \(\bp \in \VV^*\) and \(\alpha \in \RR\) such that

    \[ \langle \bx , \bp \rangle \leq \alpha < \langle \by, \bp \rangle \Forall \bx \in B. \]

  5. Taking supremum over \(B\) on the L.H.S., we obtain

    \[ \sigma_B(\bp) \leq \alpha < \langle \by, \bp \rangle \leq \sigma_A(\bp). \]

  6. Thus, there exists \(\bp \in \VV^*\) such that

    \[ \sigma_B(\bp) < \sigma_A(\bp). \]

  7. This contradicts our hypothesis that \(\sigma_A\) and \(\sigma_B\) are identical.

  8. Thus, \(A = B\) must hold.

The next result shows that support function for a set and its closure or its convex hull are identical. This is why, we required \(A,B\) to be closed and convex in the previous result.

Theorem 6.85 (Support functions and closure or convex hull of underlying set)

Let \(A \subseteq \VV\). Then,

  1. \(\sigma_A = \sigma_{\closure A}\).

  2. \(\sigma_A = \sigma_{\convex A}\).

Proof. We first consider the case of closure.

  1. \(A \subseteq \closure A\).

  2. Thus, \(\sigma_A (\by) \leq \sigma_{\closure A}(\by) \Forall \by \in \VV^*\).

  3. Let us now show the reverse inequality.

  4. Let \(\by \in \VV^*\).

  5. Then, there exists a sequence \(\{ \bx_k \}\) of \(\closure A\) such that

    \[ \lim_{k \to \infty} \langle \bx_k, \by \rangle = \sigma_{\closure A} (\by). \]

  6. Now for every \(\bx_k \in \closure A\), there exists a point \(\bz_k \in A\) such that \(d(\bx_k, \bz_k) \leq \frac{1}{k}\).

  7. Thus, \(\lim_{k \to \infty} (\bz_k - \bx_k) = \bzero\).

  8. Since \(\bz_k \in A\), hence

    \[ \sigma_A(\by) \geq \langle \bz_k, \by \rangle = \langle \bx_k, \by \rangle + \langle \bz_k - \bx_k, \by \rangle. \]

  9. Taking the limit \(k \to \infty\) on the R.H.S., we obtain

    \[ \sigma_A(\by) \geq \sigma_{\closure A}(\by) + 0 = \sigma_{\closure A}(\by). \]

  10. Thus, \(\sigma_A(\by) = \sigma_{\closure A}(\by)\) must hold true.

  11. Since this is true for every \(\by \in \VV^*\), hence \(\sigma_A = \sigma_{\closure A}\).

Now, consider the case of convex hull.

  1. By definition, \(A \subseteq \convex A\).

  2. Thus, \(\sigma_A (\by) \leq \sigma_{\convex A}(\by) \Forall \by \in \VV^*\).

  3. Let \(\by \in \VV^*\).

  4. Then, there exists a sequence \(\{ \bx_k \}\) of \(\convex A\) such that

    \[ \lim_{k \to \infty} \langle \bx_k, \by \rangle = \sigma_{\convex A} (\by). \]

  5. Since \(\bx_k \in \convex A\), hence, there exists \(\bz_1^k, \dots, \bz_{n_k}^k \in A\) and \(\bt^k \in \Delta_{n_k}\) such that

    \[ \bx_k = \sum_{i=1}^{n_k} t_i^k \bz_i^k. \]

  6. By linearity of the inner product

    \[\begin{split} \langle \bx_k, \by \rangle &= \left \langle \sum_{i=1}^{n_k} t_i^k \bz_i^k, \by \right \rangle\\ &= \sum_{i=1}^{n_k} t_i^k \langle \bz_i^k, \by \rangle\\ &\leq \sum_{i=1}^{n_k} t_i^k \sigma_A(\by)\\ &= \sigma_A(\by). \end{split}\]

  7. Taking the limit \(k \to \infty\) on the L.H.S., we obtain

    \[ \sigma_{\convex A} (\by) = \lim_{k \to \infty} \langle \bx_k, \by \rangle \leq \sigma_A(\by). \]

  8. Thus, \(\sigma_A(\by) = \sigma_{\convex A}(\by)\) must hold true.

  9. Since this is true for every \(\by \in \VV^*\), hence \(\sigma_A = \sigma_{\convex A}\).

Following properties of support functions are useful in several applications.

Theorem 6.86 (Arithmetic properties of support functions)

  1. (Nonnegative homogeneity) For any nonempty set \(C \subseteq \VV\) and a vector \(\bx \in \VV\) and \(t \geq 0\),

    \[ \sigma_C (t \bx) = t \sigma_C (\bx). \]

  2. (Subadditivity) For any nonempty set \(C \subseteq \VV\) and a vector \(\bu, \bv \in \VV\),

    \[ \sigma_C(\bu + \bv) \leq \sigma_C (\bu) + \sigma_C (\bv). \]

  3. (Nonnegative scaling of the underlying set) For any nonempty set \(C \subseteq \VV\) and a vector \(\bx \in \VV\) and \(t \geq 0\)

    \[ \sigma_{t C} (\by) = t \sigma_C(\by). \]

  4. (Additivity over Minkowski sum of sets) For any two nonempty subsets \(A, B \subseteq \VV\) and \(\bx \in \VV\)

    \[ \sigma_{A + B} (\bx) = \sigma_A(\bx) + \sigma_B(\bx). \]

Proof. (1) Nonnegative homogeneity

\[\begin{split} \sigma_C (t \bx) &= \sup_{\by \in C} \langle t \bx, \by \rangle \\ &= \sup_{\by \in C} t \langle \bx, \by \rangle \\ &= t \sup_{\by \in C} \langle \bx, \by \rangle \\ &= t \sigma_C(\bx). \end{split}\]

Here, we used the fact that \(\sup\) commutes with nonnegative scalars.

(2) Subadditivity

\[\begin{split} \sigma_C (\bu + \bv) &= \sup_{\by \in C} \langle \bu + \bv, \by \rangle \\ &= \sup_{\by \in C} (\langle \bu, \by \rangle + \langle \bu, \by \rangle) \\ &\leq \sup_{\by \in C} \langle \bu, \by \rangle + \sup_{\by \in C} \langle \bv, \by \rangle \\ &= \sigma_C (\bu) + \sigma_C(\bv). \end{split}\]

(3) Nonnegative scaling of the underlying set

\[\begin{split} \sigma_{ t C} (\bx) &= \sup_{\by \in t C} \langle \bx, \by \rangle \\ &= \sup_{\by \in C} \langle \bx, t \by \rangle \\ &= \sup_{\by \in C} t \langle \bx, \by \rangle \\ &= t \sup_{\by \in C} \langle \bx, \by \rangle \\ &= t \sigma_C(\bx). \end{split}\]

(4) Minkowski sum

\[\begin{split} \sigma_{ A + B} (\bx) &= \sup_{\by \in A + B} \langle \bx, \by \rangle \\ &= \sup_{\bu \in A, \bv \in B} \langle \bx , \bu + \bv \rangle \\ &= \sup_{\bu \in A, \bv \in B} (\langle \bx , \bu \rangle + \langle \bx , \bv \rangle ) \\ &= \sup_{\bu \in A} \langle \bx , \bu \rangle + \sup_{\bv \in B} \langle \bx , \bv \rangle\\ &= \sigma_A (\bx) + \sigma_B(\bx). \end{split}\]

6.6.11. Gauge Functions¶

Definition 6.50 (Gauge function for a set)

Let \(\VV\) be a real vector space. Let \(C\) be a nonempty subset of \(\VV\). The gauge function \(\gamma_C : \VV \to \RERL\) is defined as

\[ \gamma_C (\bx) = \inf \{r \geq 0 \ST \bx \in r C \}. \]

If \(\bx \notin r C\) for any \(r \geq 0\), then \(\gamma_C(\bx) = \infty\). This is consistent with the convention that \(\inf \EmptySet = \infty\).

The gauge function is also known as Minkowski functional.

Property 6.8 (Nonnegativity)

The Gauge function is always nonnegative.

Property 6.9 (Value at origin)

\[ \gamma_C (\bzero) = 0. \]

Property 6.10 (Subadditive)

If \(C\) is convex, then the gauge function is subadditive.

\[ \gamma_C (\bx + \by) \leq \gamma_C (\bx) + \gamma_C(\by). \]

Proof. We proceed as follows.

  1. If \(\gamma_C (\bx) = \infty\) or \(\gamma_C(\by) = \infty\), then the inequality is satisfied trivially. So assume that both are finite.

  2. Then, the sets \(X = \{r \geq 0 \ST \bx \in r C \}\) and \(Y = \{r \geq 0 \ST \by \in r C \}\) are not empty.

  3. Thus, we can choose some \(s \geq \gamma_C (\bx)\) from \(X\) and \(t \geq \gamma_C(\by)\) from \(Y\).

  4. If \(s=0\) or \(t=0\), then \(\bzero \in C\). Consequently,

    \[ \gamma_C (\bx + \by) = \gamma_C (\bx) = \gamma_C (\by) = 0 \leq \gamma_C (\bx) + \gamma_C(\by) \leq s + t. \]

    and the inequality is satisfied.

  5. Now, consider the case where \(s > 0\) and \(t > 0\).

  6. Then, \(\frac{\bx}{s} \in C\) and \(\frac{\by}{t} \in C\).

  7. Now,

    \[ \frac{\bx + \by}{s + t} = \frac{s}{s + t} \frac{\bx}{s} + \frac{t}{s + t}\frac{\by}{t}. \]

  8. Thus, \(\frac{\bx + \by}{s + t}\) is a convex combination of \(\frac{\bx}{s}\) and \(\frac{\by}{t}\).

  9. Since \(C\) is convex, hence \(\frac{\bx + \by}{s + t} \in C\).

  10. Thus, \(s + t \in \{ r \geq 0 \ST (\bx + \by) \in rC \}\).

  11. Thus, \(\gamma_C (\bx + \by) \leq s + t\).

  12. Thus, for every \(s \geq \gamma_C (\bx)\) and every \(t \geq \gamma_C(\by)\), \(\gamma_C (\bx + \by) \leq s + t\).

  13. Taking infimum on the R.H.S. over \(s \in X\) and \(t \in Y\), we obtain, \(\gamma_C (\bx + \by) \leq \gamma_C (\bx) + \gamma_C(\by)\).

Property 6.11 (Homogeneous)

The Gauge function is homogeneous.

\[ \gamma_C (s \bx) = |s| \gamma_C (\bx) \Forall s \in \RR. \]

Property 6.12 (Seminorm)

The Gauge function is a seminorm.

Example 6.34 (Norm as a gauge function)

Let \(\VV\) be a normed linear space with the norm \(\| \cdot \| : \VV \to \RR\).

Let \(\overline{B} = \{\bx \in \VV \ST \| \bx \| \leq 1 \}\) be the unit closed ball.

Then,

\[ \gamma_{\overline{B}} (\bx) = \inf \{r \geq 0 \ST \bx \in r \overline{B} \} = \| \bx \|. \]

The gauge function for the closed unit ball is simply the norm itself.

6.6.12. Jensen’s Inequality¶

Jensen’s inequality stated below is another formulation for convex functions.

Theorem 6.87 (Jensen’s inequality)

A proper function \(f: \VV \to \RERL\) is convex if and only if

(6.2)¶\[f(t_1 \bx_1 + \dots + t_k \bx_k) \leq t_1 f(\bx_1) + \dots + t_k f(\bx_k)\]

holds true for every \(\bx_1, \dots, \bx_k \in \VV\) whenever \(t_1, \dots, t_k \geq 0\) and \(t_1 + \dots + t_k = 1\). This inequality is known as the Jensen’s inequality.

Proof. The Jensen’s inequality reduces to (6.1) for \(k=2\). Thus, the statement is true by definition for \(k=2\). For \(k > 2\), we shall present an inductive proof.

Assume \(f\) is convex. Then \(\dom f\) is convex and for all \(\bx_1,\bx_2 \in \dom f\), and \(t \in [0, 1]\), we have:

\[ f(t \bx_1 + (1-t) \bx_2) \leq t f(\bx_1) + (1-t) f(\bx_2). \]

  1. Let \(\bx_1, \dots, \bx_k \in \VV\).

  2. If any of \(\bx_i \notin \dom f\) for some \(i \in 1,\dots,k\), then \(f(\bx_i) = \infty\) and the Jensen’s inequality holds trivially.

  3. Thus, we shall assume that \(\bx_1, \dots, \bx_k \in \dom f\).

  4. Since \(\dom f\) is convex, hence their convex combination \(t_1 \bx_1 + \dots + t_k \bx_k \in \dom f\).

  5. Inductively, assume that the Jensen’s inequality holds for \(k-1\); i.e.,

    \[ f(r_1 \bx_1 + \dots + r_{k-1} \bx_{k-1}) \leq r_1 f(\bx_1) + \dots + r_{k-1} f(\bx_{k-1}) \]

    holds true whenever \(r_1, \dots, r_{k-1} \geq 0\) and \(r_1 + \dots + r_{k-1} = 1\).

  6. WLOG, assume that \(t_k < 1\). Thus, \(1 - t_k > 0\).

  7. Define \(\by = \sum_{i=1}^{k-1} t'_i \bx_i\) where \(t'_i = \frac{t_i}{1 - t_k}\).

  8. Note that \(t'_i \geq 0\). Also, \(\sum_{i=1}^{k-1} t'_i = 1\) since \(\sum_{i=1}^{k-1} t_i = 1 - t_k\).

  9. We can now write:

    \[\begin{split} f(t_1 \bx_1 + \dots + t_k \bx_k) &= f((1 - t_k) \by + t_k \bx_k )\\ &\leq (1 - t_k)f (\by) + t_k f(\bx_k)\\ &= (1 - t_k) f(t'_1 \bx_1 + \dots t'_{k-1} \bx_{k-1}) + t_k f(\bx_k)\\ &\leq (1 - t_k) ( t'_1 f(\bx_1) + \dots + t'_{k-1} f(\bx_{k-1})) + t_k f(\bx_k)\\ &= t_1 f(\bx_1) + \dots + t_{k-1} f(\bx_{k-1}) + t_k f(\bx_k). \end{split}\]

  10. Thus, \(f\) satisfies Jensen’s inequality.

For the converse, assume that \(f\) satisfies Jensen’s inequality. Let \(\bx_1, \bx_2 \in \dom f\) and \(t \in [0,1]\). Then, by Jensen’s inequality for \(k=2\),

\[ f(t \bx_1 + (1-t) \bx_2) \leq t f(\bx_1) + (1-t) f(\bx_2) < \infty. \]

Thus, \(t \bx_1 + (1-t) \bx_2 \in \dom f\). Thus, \(\dom f\) is convex. Also, \(f\) satisfies (6.1). Hence, \(f\) is convex.

Jensen’s inequality is essential in proving a number of famous inequalities.

Example 6.35 (Logarithm and Jensen’s inequality)

In Example 6.41, we show that \(\ln(x)\) is concave. Consequently, \(-\ln(x)\) is convex.

Now, let \(x_1, \dots, x_n \in \RR_{++}\) be positive real numbers and let \(t_1, \dots, t_n \geq 0\) such that \(t_1 + \dots + t_n = 1\). Then, by Jensen’s inequality

\[ -\ln (t_1 x_1 + \dots + t_n x_n) \leq - t_1 \ln x_1 - \dots - t_n \ln x_n. \]

Multiplying by \(-1\) and taking exponential on both sides, we obtain

\[ t_1 x_1 + \dots + t_n x_n \geq x_1^{t_1} \dots x_n^{t_n}. \]

For a particular choice of \(t_1 = \dots = t_n = \frac{1}{n}\), we obtain

\[ \frac{1}{n}(x_1 + \dots + x_n) \geq \sqrt[n]{x_1 \dots x_n} \]

which is the AM-GM inequality suggesting that arithmetic mean is greater than or equal to the geometric mean for a group of positive real numbers.

Theorem 6.88 (Jensen’s inequality for nonnegative homogeneous convex functions)

If \(f: \VV \to \RERL\) is a nonnegative homogeneous proper convex function, then

(6.3)¶\[f(t_1 \bx_1 + \dots + t_k \bx_k) \leq t_1 f(\bx_1) + \dots + t_k f(\bx_k)\]

holds true for every \(\bx_1, \dots, \bx_k \in \VV\) whenever \(t_1, \dots, t_k \geq 0\).

Proof. Let \(\bx_1, \dots, \bx_k \in \VV\). If any of \(\bx_i \notin \dom f\) for some \(i \in 1,\dots,k\), then \(f(\bx_i) = \infty\) and the inequality holds trivially. Thus, we shall assume that \(\bx_1, \dots, \bx_k \in \dom f\).

By Theorem 6.71, \(f\) is subadditive. Thus,

\[ f(t_1 \bx_1 + \dots + t_k \bx_k) \leq f(t_1 \bx_1) + \dots + f(t_k \bx_k). \]

The nonnegative homogeneity gives us

\[ f(t_1 \bx_1) + \dots + f(t_k \bx_k) = t_1 f(\bx_1) + \dots + t_k f(\bx_k). \]

We are done.

6.6.13. Convex Real Functions¶

For real functions \(f : \RR \to \RR\), several useful results are available.

Theorem 6.89 (Characterization of real convex functions)

Let \(f: \RR \to \RR\) be a real function with \(\dom f = (a,b)\) which is an open interval. The following are equivalent.

  1. \(f\) is convex.

  2. For every \(x_1, x_2, x_3 \in (a, b)\) with \(x_1 < x_2 < x_3\),

    \[ \frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1}. \]

  3. For every \(x_1, x_2, x_3 \in (a, b)\) with \(x_1 < x_2 < x_3\),

    \[ \frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_2)}{x_3 - x_2}. \]

Proof. (1) \(\implies\) (2) Assume that \(f\) is convex.

  1. Let

    \[ \alpha = \frac{x_3 - x_2}{x_3 - x_1}, \beta = \frac{x_2 - x_1}{x_3 - x_1}. \]

  2. Then, \(\alpha + \beta = 1\) and \(\alpha , \beta \in (0,1)\).

  3. Also, verify that

    \[ x_2 = \alpha x_1 + \beta x_3. \]

  4. Thus,

    \[\begin{split} & f(x_2) \leq \alpha f(x_1) + \beta f(x_3) \\ & \iff f(x_2) \leq \frac{x_3 - x_2}{x_3 - x_1} f(x_1) + \frac{x_2 - x_1}{x_3 - x_1} f(x_3)\\ & \iff (x_3 - x_1) f(x_2) \leq (x_3 - x_2) f(x_1) + (x_2 - x_1) f(x_3)\\ & \iff (x_3 - x_1) f(x_2) \leq ((x_3 - x_1) - (x_2 - x_1)) f(x_1) + (x_2 - x_1) f(x_3)\\ & \iff (x_3 - x_1) (f(x_2) - f(x_1)) \leq (x_2 - x_1) (f(x_3) - f(x_1))\\ & \iff \frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1}. \end{split}\]

(2) \(\implies\) (1)

  1. Let \(x_1, x_3 \in (a, b)\) and \(t \in (0, 1)\).

  2. WLOG, assume that \(x_1 < x_3\).

  3. Let \(\alpha = t\) and \(\beta = (1-t)\).

  4. Let \(x_2 = \alpha x_1 + \beta x_3\).

  5. Then, \(x_1 < x_2 < x_3\).

  6. From the hypothesis, we have

    \[ \frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1}. \]

  7. Using the previous argument backwards, this implies

    \[ f(x_2) \leq \alpha f(x_1) + \beta f(x_3) = t f(x_1) + (1-t) f(x_3). \]

  8. Thus, \(f\) is convex.

We can prove the rest similarly.

Theorem 6.90 (Real convex functions and one sided derivatives)

Let \(f: \RR \to \RR\) with \(\dom f = (a,b)\) be convex. Then, for every \(x \in (a,b)\), the left hand derivative \(f'_-(x)\) and the right hand derivative \(f'_+(x)\) exist.

Proof. We are given that \(f\) is convex over \((a,b)\).

  1. Let \(x \in (a,b)\).

  2. For \(h > 0\), define

    \[ F(h) = \frac{f(x + h) - f(x)}{h}. \]

  3. Let \(0 < h_1 < h_2\) such that \(x + h_2 < b\).

  4. Let \(x_1 = x\), \(x_2 = x + h_1\), \(x_3 = x + h_2\).

  5. Since \(f\) is convex, hence by Theorem 6.89

    \[ \frac{f(x_2) - f(x_1)}{x_2 - x_1} \leq \frac{f(x_3) - f(x_1)}{x_3 - x_1}. \]

  6. But that means

\[ \frac{f(x + h_1) - f(x)}{h_1} \leq \frac{f(x + h_2) - f(x)}{h_2}. \]

  1. Thus, whenever \(h_1 < h_2\) (up to \(h_2 < b - x\)), \(F(h_1) \leq F(h_2)\).

  2. Thus, \(F\) is a nondecreasing (monotone) function of \(h\) in some interval \((0, \delta)\) where \(\delta < b - x\).

  3. Then, \(f'_+(x) = \lim_{h \to 0^+} F(h)\) exists.

A similar argument shows that \(f'_-(x)\) also exists.

Theorem 6.91 (Convex real function is continuous)

Let \(f: \RR \to \RR\) with \(\dom f = (a,b)\) be convex. Then, for every \(x \in (a,b)\), Then, \(f\) is continuous on \((a,b)\).

Proof. We proceed as follows.

  1. Let \(x \in (a,b)\).

  2. By Theorem 6.90, the one sided derivatives \(f'_+(x)\) and \(f'_-(x)\) exist.

  3. Then, by limit arithmetic

    \[ \lim_{h \to 0^-} (f(x + h) - f(x)) = \left ( \lim_{h \to 0^-} \frac{f(x + h) - f(x)}{h} \right ) \left ( \lim_{h \to 0^-} h \right) = 0. \]

  4. Similarly,

    \[ \lim_{h \to 0^+} (f(x + h) - f(x)) = \left ( \lim_{h \to 0^+} \frac{f(x + h) - f(x)}{h} \right ) \left ( \lim_{h \to 0^+} h \right) = 0. \]

  5. Thus,

    \[ \lim_{h \to 0^-} (f(x + h) - f(x)) = \lim_{h \to 0^+} (f(x + h) - f(x)) = 0. \]

  6. Thus, \(f\) is continuous at \(x\).

  7. Since \(x\) was arbitrary, hence \(f\) is continuous on \((a,b)\).

6.6.14. Quasi-Convex Functions¶

Definition 6.51 (Quasi convex function)

Let \(\VV\) be a real vector space. Let \(f : \VV \to \RR\) be a real valued function. Let the sublevel sets of \(f\) be given by

\[ C_{\alpha} = \{ \bx \in \dom f \,|\, f(\bx) \leq \alpha \}. \]

If the sublevel sets \(C_{\alpha}\) of \(f\) are convex for every \(\alpha \in \RR\), then \(f\) is called a quasi-convex function.
